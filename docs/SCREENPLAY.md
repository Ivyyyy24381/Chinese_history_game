## 编剧体例说明

每个事件按下列结构展开，写完即可对照转写成 `event.json`：

- **设计意图**：一句话说本事件在整条人物弧线里负责什么。
- **Phase 列表**：每个 phase 标注 `type` —— `explore` / `exam` / `transition` / `forced_choice` / `poem_compose` / `map_travel`。
- **每个 phase 给出**：
  - `背景`：建议的背景图（资源命名 `/assets/events/<id>/backgrounds/<name>.png`）
  - `叙事 narrative`：开场旁白（可为空）
  - `指示 instruction`：给玩家的操作提示
  - `NPC / 可点击物件`：ID、名称、立绘、位置（百分比坐标 0–100）、对话；标 `🔑` 表示该项是必看「线索」。
  - `通过条件`：达成 N 次交谈 / 全部线索 / 选择正确等才能进入下一 phase。
  - `nextPhase` 或 `triggers`：跳转。
- **诗作**：与本事件绑定的诗，写明出处与节选。
- **资源需求**：背景图、专属 NPC 立绘清单。

Phase 类型约定（与 `ScenePlayer.jsx` 已支持的一致）：

| type | 用途 | 关键字段 |
|---|---|---|
| `explore` | 自由探索点 NPC | `npcs[]`、`requiredTalks`、`triggers[]` |
| `exam` | 出题考试 | `examiner`、`questions[]`（`choice` / `poem_fill`） |
| `transition` | 过场 / 字幕 / 诏书 | `transitionText`、`announcement`、`dufu_reaction` |
| `forced_choice` | 单选题，分支台词 | `question`、`options[]`、`conclusion` |
| `poem_compose` | 拼诗 / 补诗 | `prompt`、`tokens[]`、`answer` |
| `map_travel` | 在大地图上移动 | `from`、`to`、`waypoints` |

## 744 遇李白

**地点**：洛阳 → 梁宋 → 齐鲁　**时期**：少年壮游　**状态**：浪漫理想
📝 草稿，待审阅后实现

### 设计意图

杜甫一生最快意的一年。让玩家用「酒、诗、马」三件道具把握李白这个浪漫人物，并在最后一次「送别」做出小选择——选择不会改变剧情，但会决定杜甫一生在李白题材上写哪一首怀念诗。建立「理想未碎前的样子」。

### Phase 1 · explore · 洛阳酒肆——听闻李白

- **背景**：`luoyang_jiusi.png`（酒肆内，胡姬、酒坛、压角的灯笼）
- **叙事**：「天宝三载（744），春。洛阳市井疯传一个名字——李白。」
- **指示**：「在酒肆里转一圈，听人怎么说李白。」
- **NPC**：
  - `drinker_a` 酒客甲（立绘 `drinker_a.png`）位置 (25, 60)
    - 「听说了吗？李太白被赐金放还，从长安出来了！」
    - 「赐金放还——好听是好听，说白了就是被打发走了。」
  - `drinker_b` 酒客乙（立绘 `drinker_b.png`）位置 (55, 65)
    - 「就是那个『天子呼来不上船，自称臣是酒中仙』的李白？」
    - 「啧。这种人，敢死。」
  - `huji` 胡姬（立绘 `huji.png`）位置 (75, 55)
    - 「这位郎君要点什么？西市新到的葡萄酒？」
    - 「李太白？前日才来过。喝完三斗就走了，留了一首诗压账。」
    - 🔑（点击「李白留下的诗」物件）→ 弹出诗笺：「兰陵美酒郁金香，玉碗盛来琥珀光……」
  - 🔑 `dufu_inner` 杜甫·自语（无立绘——是一个内心独白触发点）位置 (45, 35)
    - 「李太白……我读他的诗，已读了十年。」
    - 「若能见他一面……」
- **通过条件**：与 3 人交谈 + 必看 `dufu_inner` → `nextPhase: street_meet`

### Phase 2 · forced_choice · 街头相遇（小破冰）

- **背景**：`luoyang_street.png`（市集，对面骑驴而来一披白裘的中年人）
- **叙事**：「转过街角，一个身披白裘、腰悬长剑的中年人正在驴上仰头大笑。」
- **题**：你怎么搭话？
  - A「在下巩县杜甫，字子美。久仰太白先生大名。」→ **正确**（建立尊重）
    - 李白：「哈！子美？你的『骑驴十三载』我也读过。」
  - B「太白先生！可愿与小弟共饮一杯？」→ **正确**（建立放浪）
    - 李白：「妙！同道中人，何须客套？」
  - C 默默走开 → **错误**
    - 旁白：「擦肩而过的那一刻，杜甫心里千言万语，却一句也说不出。」
    - （回到选择）
- **结论**：「两人一见如故。这是天宝三载春，洛阳。」→ `nextPhase: liangsong_tour`

    - 杜甫：「太白，再留半月可好？」

- **结论**：「此一别，两人再未相见。但杜甫一生写了十余首怀李白的诗。」→ `nextPhase: ending`

## 747 制举「野无遗贤」

## 751 进三大礼赋

**地点**：长安　**时期**：困守长安　**状态**：开始向权力妥协
✅ 已实现（interactive，7 phases）

> 此事件已写成完整 `event.json`。背景资源现已对应到：`changan_street.png` (借 747)、`dufu_home.png`、`yanen_gate.png`、`jixianyuan.png`、`changan_night.png`。

### 设计意图

### Interactive 设计的几条共通原则

1. **「3 NPC + 1 物件」是默认的探索拍号**——既不冷场也不疲劳。
2. **🔑 线索一定要有不同于普通对话的视觉提示**（描边、轻微浮动），否则玩家容易漏看。
3. **forced_choice 的「错答」最好不是 game over**，而是给一句嘲讽 + 回到选择，让玩家自然选到「对」的那一条。
4. **每个事件至少要有一首诗的拼接 / 揭示动作**——这是"教育"卖点。
5. **transition 的 announcement 应该是有「字幕字体感」的画面**（书法 / 诏书 / 题字），不要混在普通对话框里。
