# 怎么加背景音乐 · ADD_MUSIC

> 引擎已内置 BGM 系统：**一个时期一首循环曲**，按时间轴所在时期自动切歌。
> 你只需要把文件按命名放进文件夹，零代码。

## 1. 文件放哪、叫什么

路径：`public/assets/audio/bgm/<stageId>.mp3`

stageId 来自人物 `timeline.json` 的 stages[].id。杜甫线的五个：

| 文件名 | 时期 | 年份 | 建议风格 |
|---|---|---|---|
| `youth.mp3` | 少年壮游 | 712—746 | 明亮古琴/笛，意气风发 |
| `changan.mp3` | 困守长安 | 746—755 | 沉郁古筝，都市底色 |
| `anshi.mp3` | 安史烽火 | 755—759 | 鼓点紧张，战乱悲怆 |
| `chengdu.mp3` | 成都草堂 | 759—765 | 田园舒缓，雨声竹林 |
| `piaobo.mp3` | 江湖漂泊 | 765—770 | 箫声苍凉，水声孤舟 |

## 2. 文件要求

- 格式 mp3，**1–3 分钟、可无缝循环**（结尾别有渐弱长尾，循环会断气口）
- 体积 ≤ 2MB/首（导出 128kbps 足够）
- 不用调音量，游戏统一播 0.35

## 3. 行为说明（已实现，无需配置）

- 进入游戏后随时间轴自动切换对应时期的曲子，循环播放
- **缺哪个文件哪个时期就静音**，不报错——可以一首一首慢慢补
- 右下角 🎵/🔇 按钮开关，玩家偏好记在 localStorage
- 浏览器自动播放限制：玩家在选人页点过鼠标后才会出声（正常流程必然满足）

## 4. 音乐从哪来

- AI 生成：Suno / 网易天音 / Stable Audio，prompt 直接用上表「建议风格」+「中国古风 器乐 无人声 可循环」
- 免版权曲库：Pixabay Music、爱给网（标注可商用）——注意保留授权凭证
- 自委托作曲：把上表风格描述发给作曲者即可

## 5. 验证

```bash
npm run dev
# 进游戏 → 拖时间轴跨时期 → 听切歌；点 🎵 关闭 → 刷新仍是关闭状态
```

## 6. 新人物怎么加

查该人物 timeline.json 的 stage id，按 `<stageId>.mp3` 命名放进同一文件夹即可。
（不同人物 stage id 不要重名，建议加前缀，如 `libai_shu.mp3`。）
