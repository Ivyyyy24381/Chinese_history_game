# 背景音乐 (BGM)

每个时期一首循环背景乐。把 mp3 文件按下列名字放进本目录即可生效，
**缺哪个文件哪个时期就静音，游戏不会报错**。

| 文件名 | 时期 | 年份 | 建议风格 |
|---|---|---|---|
| `youth.mp3` | 少年壮游 | 712—746 | 明亮古琴/笛，意气风发 |
| `changan.mp3` | 困守长安 | 746—755 | 沉郁古筝，都市底色 |
| `anshi.mp3` | 安史烽火 | 755—759 | 鼓点紧张，战乱悲怆 |
| `chengdu.mp3` | 成都草堂 | 759—765 | 田园舒缓，雨声竹林 |
| `piaobo.mp3` | 江湖漂泊 | 765—770 | 箫声苍凉，水声孤舟 |

要求：
- 格式 mp3（或改 `App.jsx` 里的扩展名），建议 1—3 分钟可循环、≤ 2MB
- 音量在游戏里统一设为 0.35，不用自己压
- 玩家可用右下角 🎵 按钮开关，偏好会记住

新增人物时：在对应人物的 timeline.json 里查 stage id，按 `<stageId>.mp3` 命名放入。
